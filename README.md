# simple-highlighter-chrome-extension

This repository is licensed under MIT-license.

## Description
A chrome extension using manifest v3 with Javascript. Main objective is to allow user to highlight a series of string from a set of chrome tab, integrated a side panel to retrive and review the highlighted string from corresponding website, effectively booasting the reading speed to comprehend the content.

## Why I develop this chrome extension
I use it to help reading research article. Main idea is to use color to associate the multiple set of token which I find interesting or useful, eventually facilitating the efficiency on understanding the context.
